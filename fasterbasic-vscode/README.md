# FasterBASIC for VS Code

Language support for FasterBASIC, a high-performance compiled BASIC dialect using the QBE backend.

## Features

- **Syntax Highlighting**: Complete highlighting for keywords, types, and operators.
- **Snippets**: Common code blocks for Loops, Control Flow, and Type definitions.
- **Problem Matching**: Automatic error detection and underlining when building.

## Setup

1. Open your FasterBASIC project in VS Code.
2. The extension automatically detects `.bas`, `.bi`, and `.fbas` files.

## Building and Running

This workspace includes a pre-configured Task Runner.

- Press `Cmd+Shift+B` (or `Ctrl+Shift+B`) to **Build** the current file.
- Or use `Terminal > Run Task... > FasterBASIC: Run Current File` to build and execute.
