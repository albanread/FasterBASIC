────────────────────────────────────────────────────────────────────────
WORKING COMMANDS
────────────────────────────────────────────────────────────────────────

CLS                    Clear screen and home cursor
GCLS                   Graphics clear (alias for CLS)
LOCATE row, col        Position cursor (1-based coordinates)
COLOR fg               Set foreground color (0-15)
COLOR fg, bg           Set foreground and background colors

Example:
   CLS
   LOCATE 10, 20
   COLOR 14, 1
   PRINT "Yellow on blue!"
